# Coalitions
* Load this lua, then talk to a Task Delegator to be able to select any coalition assignment regardless of your rank.
    * This lets you do every one exactly once, which will get you the Shovel and Cordon KIs.

* Golden Shovel Cordon KI grants access to The Ygnas Directive, therefore the Ygnas trust and +1 Adoulin rings.
    * You still need to actually be Legend in all coalitions to start making <b>Epeo or Idris</b>, even with this.
    * However, getting all of the cordon KIs will significantly accelerate the Legend process by lowering the imprimatur cooldown as well.
